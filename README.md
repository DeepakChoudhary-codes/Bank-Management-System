# Bank Management System

## Description
A console-based banking application developed in Python using Object-Oriented Programming. The application allows users to manage bank accounts and perform basic banking operations.

## Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Transaction History
- File Handling for Data Storage

## Technologies Used
- Python
- Object-Oriented Programming (OOP)
- File Handling

## How to Run
python bank_management.py
